# 💪 Get Fit - Workout App

Get Fit is a Flutter-based workout application designed to help users stay active and improve their fitness through guided exercises and structured workout plans — all without requiring an internet connection or database.

## 🚀 Features

- 🏋️‍♂️ Home & gym workout plans
- 📅 30-day fitness programs
- 🎯 Target-based workouts (Chest, Arms, Abs, Legs, etc.)
- ▶️ Step-by-step exercise guidance
- ⏱️ Built-in timers for workouts
- 🔊 Voice/Text instructions support
- 📊 Simple and clean UI for easy navigation
- 📱 Offline access (No database required)

## 📱 App Purpose

The app is built for users who want a simple and effective way to follow workout routines without needing an account or internet connection.

## 🧠 State Management

- BLoC (Business Logic Component)
- Clean separation of UI and business logic
- Scalable and maintainable architecture

## 🛠️ Tech Stack

- Flutter (Dart)
- BLoC State Management
- Local assets (images, exercise data)

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/get-fit-app.git
```

2. Navigate to project:
```bash
cd get-fit-app
```

3. Get dependencies:
```bash
flutter pub get
```

4. Run the app:
```bash
flutter run
```
